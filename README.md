## Project Description
This project is a Task Manager application built for the Android Mobile Development course. The app allows users to add tasks, view them in a list, mark them as complete, and delete them. It demonstrates the use of Activities, Intents, ListView, Custom Adapters, and basic UI components in Android development using Java.

## How to Run the Project

### 1. System Requirements
Before running this project, ensure your computer has the following installed:
- Android Studio (Latest stable version)
- Java Development Kit (JDK) version 11 or higher
- Android SDK with (Android 9.0) or higher
- Git

### 2. Installation Instructions
1. Open your command prompt or terminal.
2. Navigate to the folder where you want to save the project.
3. Clone the repository by typing the following command and pressing Enter:
   ```
   git clone https://github.com/md-maksudur-rahman-mahi/mobile-development.git
   ```
4. Wait for the download to complete.

### 3. Opening the Project
1. Launch Android Studio.
2. Click on **File** in the top menu bar.
3. Select **Open** from the dropdown menu.
4. Navigate to the folder you just cloned.
5. Open the **Project** folder, then select the **mobile_app** folder.
   - Note: You must select the folder that contains the `build.gradle` file inside the `app` directory.
6. Click **OK**.
7. Wait for Gradle to sync. This may take a few minutes depending on your internet connection.

### 4. Running the Application
1. Connect an Android phone to your computer via USB cable.
   - Ensure **USB Debugging** is enabled on your phone.
   - OR: Create an Android Virtual Device (Emulator) by clicking the Device Manager icon in Android Studio.
2. Select your device from the device dropdown menu at the top of Android Studio.
3. Click the green **Run** button (Play icon) in the toolbar.
4. Wait for the app to build and install on your device.
5. The app will open automatically once installation is complete.

### 5. Testing the Features
Once the app is open, follow these steps to verify all features are working:
1. **Add a Task:** Type text into the input field and click the **Add Task** button. A message should appear confirming the task was added.
2. **View List:** Click the **View List** button. The second screen should open showing your tasks.
3. **Complete a Task:** Click the checkbox next to a task. The text should become crossed out.
4. **Delete a Task:** Click the **Delete** button next to a task. The task should disappear from the list.
5. **Clear All:** Click the **Clear All Tasks** button at the bottom. All tasks should be removed.
